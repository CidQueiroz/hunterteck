# Frontend - Streamlit Command Center

## Overview

Streamlit-based visual interface for HunterTeck pipeline.

## App: `app_hunter.py`

Command Center for B2B prospecting:
- Campaign selection
- Lead volume control  
- Real-time pipeline execution
- Results visualization
- CSV export

## Installation

```bash
pip install -e ".[streamlit]"
```

## Running

```bash
streamlit run app_hunter.py
```

Visit: http://localhost:8501
