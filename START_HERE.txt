✅ 5 MICROSSERVIÇOS PRINCIPAIS

  🔵 MS1: Lead Extractor
     Coleta até 2,000 leads/mês de múltiplas fontes
     • Google Maps API
     • Web Scraping (BeautifulSoup)
     • APIs customizadas
     • Demo mode (sem custos)
     📂 Arquivo: services/lead_extractor/main.py (extrator + orquestrador básica)
                 services/lead_extractor/extractors.py (strategies)

  🟢 MS2: Data Validator
     Valida e deduplica dados coletados
     • Email validation (padrão corporativo)
     • Website validation
     • Detecção de duplicatas
     • Score de qualidade (0-100)
     • Limpeza de dados
     📂 Arquivo: services/lead_extractor/validator.py

  🟡 MS3: Data Enricher
     Coleta informações adicionais sobre empresas
     • Receita anual estimada
     • Número de funcionários
     • Setor/Indústria
     • URLs de redes sociais
     • Ano de fundação
     📂 Arquivo: services/lead_extractor/enricher.py

  🟣 MS4: Person Finder
     Localiza decisores (CEOs, CTOs, Diretores)
     • Encontra múltiplos roles por empresa
     • Gera emails corporativos realistas
     • Score de confiança por email
     • Nível hierárquico
     📂 Arquivo: services/lead_extractor/person_finder.py

  🔴 MS6: Email Generator
     Gera emails personalizados com IA
     • Mode Template (rápido, sem custos)
     • Mode OpenAI GPT-4 (mais personalizado)
     • A/B testing automático
     • Múltiplas sequências (first contact, followups)
     📂 Arquivo: services/lead_extractor/email_generator.py


✅ ORQUESTRADOR CENTRAL

  🎼 Pipeline Autônomo B2B
     Coordena MS1-MS7 em fluxo automático
     • Execução sequencial com error handling
     • Logging estruturado
     • Relatórios em tempo real
     📂 Arquivo: orquestrador.py
